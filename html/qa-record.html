Git

Git 下载代码提示SSL certificate problem: self signed certificate in certificate chain解决方法
git config --global http.sslVerify false
git merge了错误分支，如何优雅的回退到merge前的状态?
如果没有push的话，可以执行git reset --hard 'commit id', 清空暂存区，销毁数据，如果没有推送到远程，数据就会被覆盖无法恢复，如果已推送远程，可以通过 reflog 找回。

CSS

CSS中input输入框，在定义border的情况下，如果border-color定义为transparent,会导致border线条显示不出来， 出现input在获得焦点的时候边框缩小
border: 1px solid #ced4da;
background-color: #ffffff; /*不能定义transparent*/
CSS中:before 等选择器里面的content属性标识的内容是属于:before作用于的元素的
在每一段文字前面通过CSS自动增加标识的时候，我们需要把:before的样式设置成 position: absolute;,从当前文档流 中拉出来 
    .blog-write-main .blog-qa-list .blog-qa .blog-q:before{
        counter-increment: qa;
        content: counter(qa)'.Q:' ;
        font-size:1.3rem;
        font-weight: 700;
        padding-right: 0.5rem;
        margin-left: -4rem;
        position: absolute;
        line-height: 1.5rem;
        min-width:4rem;
        text-align: right;
    }

Mybatis

statement not found.
这种情况下，首先检查mapper scan有没有扫描到想要的mapper定义， 第二，如果你的statement是在xml里面定义的，检查mapper-locations定义的路径下是否存在xml定义， 并确保定义的文件在classes路径中存在，即能从classpath里面扫描到。
如果使用idea工具，请确认target/classes (MAVEN) 路径下是否存在xml文件(statement sql定义在文件中)
