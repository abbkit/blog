Spring Boot Jar启动方式

nohup java -jar -Dspring.profiles.active=aliyun -DLEMON_FINAL_LOG_DIR=/work/blog-api/log blog-api-starter-3.0.1.jar 2>&1 >/dev/null &

  • -D 指定系统启动参数、环境变量等

Log4j XML 动态参数

<Property name="rollingDir">${sys:LEMON_FINAL_LOG_DIR}</Property>

  • 环境变量可以通过JVM启动参数-D设置

@media 查询 我们在需要根据device的宽高动态定义某个div或者页面元素的宽高时。可以使用 calc , min-height: calc(100vh - 180px);

  • vh -> viewport height (百分比,比如 1vh=1% ， 100vh=100%)
  • vw -> viewport width (百分比,比如 1vw=1% ， 100vw=100%)
